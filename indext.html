Seorang Android Development adalah seorang software developer yang bertugas merancang dan membangun aplikasi untuk marketplace Android. Jenkd program yang dibuat mencakup aplikasi dan juga game, baik untuk ponsel cerdas Android ataupun tablet. B Dan untuk bahasa pemrograman yang digunakan biasanya bahasa pemrograman Kotlin, Java, C/C++, dan JavaScript atau yang lainnya.

Web Developer

Web development adalah sebuah teknologi pengembangan sebuah web untuk word wide web. Kemudian untuk web developer bisa dibilang sebagai seseorang yang bertanggung jawab dalam pembuatan dan pengelolaan website. Pengembangan web yang dimaksud ini memiliki ruang lingkup e-commerce dengan cakupan pengembangan bisnis, server side, client side, scripting, web server konfigurasi, hingga desain web.

Machine Learning

Machine learning merupakan sebuah ilmu yang mempelajari tentang algoritma dan model statistik yang digunakan oleh sistem komputer untuk melakukan task tertentu tanpa instruksi eksplisit. Machine learning sangat bergantung pada pola dan kesimpulan. Untuk mendapatkan pola dan kesimpulan tersebut, algoritma machine learning menghasilkan model matematika yang didasari dari data sampel yang sering disebut dengan ‘training data.’
